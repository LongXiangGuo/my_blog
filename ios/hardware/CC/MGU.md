# MGU

**MGU：Media** Graphics Unit（宝马主流的高级多媒体娱乐主机中央大脑）。
**IDC**：Infotainment Domain Controller（新一代座舱娱乐域控制器）。

```

用蓝牙/缓存做引子 
 Wi-Fi 建立物理大管道 
 登录初始化 
 对称加密优先（ACL） 
 失败则自动升级为非对称密码学大招（JWT/G2） 
 交换证书并锁入硬件钥匙串 
 彻底开启远控大网关。

```

|                      第 0 步：经典的 蓝牙 EA（External Accessory）秘密接头               |

                                            │

                                            │
